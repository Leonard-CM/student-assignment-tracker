# Reflection – Assignment 12

Assignment 12 focused on implementing a Service Layer and REST API architecture using Spring Boot.

A key learning outcome was understanding layered architecture and the separation of concerns between repositories, services, and controllers. The service layer improved maintainability by isolating business logic from API endpoints.

Implementing RESTful APIs provided practical experience in endpoint design, HTTP methods, and backend communication. Swagger/OpenAPI documentation also demonstrated the importance of API usability and developer-focused documentation.

Testing the services and APIs reinforced the importance of validation, reliability, and maintainable enterprise application development.
