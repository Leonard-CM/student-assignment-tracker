# Kanban Board Explanation

## What is a Kanban Board?

A Kanban board is a visual project management tool used to track tasks across different stages of a workflow. It helps teams manage work efficiently by visualizing progress and limiting work-in-progress.

---

## Workflow Visualization

The Kanban board uses the following columns:

- **To Do**: Tasks that are not yet started
- **In Progress**: Tasks currently being worked on
- **Testing**: Tasks under verification
- **Blocked**: Tasks that cannot proceed due to issues
- **Done**: Completed tasks

This structure provides clear visibility into task progress.

---

## Work-In-Progress Limits

The number of tasks in the "In Progress" column is limited to 2–3 tasks at a time.

This ensures:
- Better focus
- Reduced multitasking
- Faster completion
- Avoidance of bottlenecks

---

## Agile Principles Support

The Kanban board supports Agile principles by:

- Enabling continuous delivery
- Allowing flexibility for changes
- Providing real-time progress visibility
- Supporting incremental development

This aligns with the Agile approach used in previous assignments.
