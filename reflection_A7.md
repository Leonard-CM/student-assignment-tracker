# Reflection

One of the main challenges in this assignment was selecting the most appropriate GitHub template. While multiple templates were available, it required careful evaluation to choose one that aligns with Agile principles and project requirements.

Customizing the Kanban board also presented challenges, particularly when deciding which additional columns to include. Adding "Testing" and "Blocked" improved workflow clarity but required a clear understanding of the development lifecycle.

Another challenge was linking issues and assigning tasks correctly. This highlighted the importance of traceability and proper task management in Agile environments.

Comparing GitHub Projects to tools like Trello and Jira showed that GitHub provides strong integration with development workflows, while other tools offer more advanced visualization features.

Overall, this assignment improved my understanding of Agile project management and the practical use of Kanban boards in real-world scenarios.
